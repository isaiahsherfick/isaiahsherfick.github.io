PYTHON MISSION 1

Alright. This is gonna be a leap. We're gonna try to learn Python. Python is a programming language that has skyrocketed in popularity in recent years due to its gracefully simple syntax. (That means it's really easy to read and write). Just for starters, we're going to write a "Hello World!" program. This program does one simple thing: it prints "Hello World!" to the console. This is gonna be scary, I know, but I'll hold your hand the whole way.

  1. Open your terminal by hitting ctrl+alt+t on the keyboard.
  2. (Only do this one time) Create a new directory(folder) by inputting "mkdir python" and hitting enter.
  3. Go into your new python directory by inputting "cd python" and hitting enter.
  4. Now that you're in your python directory, input "nano helloworld.py." This will open the nano text editor and create a new file, helloworld.py.
  5. In nano, type in " print("Hello World!") ". This is the only line of code you need for this program.
  6. Now, hit ctrl+o followed by enter to save the file, then hit ctrl+x to exit nano.
  7. After exiting nano, you should be in your python directory still. Run the command "python3 helloworld.py". This should run your code.
  8. If you did everything correctly, "Hello World!" should be printed to the terminal.
